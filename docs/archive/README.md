# Archive-only documents

Everything in this folder is retained for historical context, migration notes, or reference archaeology.

- These files are **not** the canonical source of truth for current Borg behavior.
- Prefer the root-level docs (`AGENTS.md`, `ARCHITECTURE.md`, `ROADMAP.md`, `CHANGELOG.md`) plus `tasks/active/` when deciding what Borg should do now.
- If an archived document conflicts with a root-level canonical doc, the root-level canonical doc wins.
