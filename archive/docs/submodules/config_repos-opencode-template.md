# OpenCode Project

Professional OpenCode project configuration and instruction set.

## Project Structure

- `agent/`: Specialized agent definitions for various engineering roles.
- `command/`: Custom slash commands for enhanced productivity.
- `instructions/`: Core development guidelines and identity definitions.
- `plugin/`: Custom plugins and tool extensions.
- `skills/`: Reusable skill sets for complex automation tasks.
- `superpowers/`: Core framework extensions.

## Setup

1. Install OpenCode CLI.
2. Clone this repository into your OpenCode config directory.
3. Update `opencode.jsonc` with your specific preferences and API keys.

## Features

- **Multi-Agent Orchestration**: Specialized agents for backend, frontend, security, and architecture.
- **Advanced Commands**: Custom workflows for SEO, debugging, and code planning.
- **Comprehensive Instructions**: Tailored coding preferences for TypeScript and general development.
- **Skill System**: Pre-built logic for common engineering challenges.
