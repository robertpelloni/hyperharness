# IDEAS: Renderer (UI/UX)

## 1. Visual Command Orchestrator

A visual timeline showing concurrent agent tasks. Drag and drop to re-order, pause, or link task outputs (e.g. "Pipe output of Coder Agent into QA Agent").

## 2. Dynamic Documentation Graph

Expand the `DocumentGraph` from static nodes to interactive "Live" nodes that show real-time agent activity (e.g. "Agent X is currently editing this file").

## 3. Keyboard Shortcut Editor

A visual GUI for mapping arbitrary keys to slash commands or internal Maestro actions, moving away from hardcoded listener maps.

## 4. "Focus Mode" HUD

A minimal, translucent overlay that stays on top of your IDE (VS Code/Cursor), showing only the current agent progress and "Stop" button, so you don't have to switch windows to check status.

## 5. Rich Data Visualizers

Native renderers for CSV, JSON, and Log files. If an agent produces a CSV, show a sortable grid or a chart instead of raw text.
